20160807
===============
完成收发测试。STM32与CAN Nalyst-II双向通信成功。
---TX----------
串口上位机向STM32发送1字节数据，格式“Tx:x” x='1','2','3' 。
STM32会将数据通过CAN总线发送出去，同时控制自身相应的LED切换状态
---Rx----------
STM32接收到数据，进入中断USB_LP_CAN1_RX0_IRQHandler,辨别ID等数据。
如果是正确数据包，将根据接收到的数据，切换自身LED状态。
